# 世界地図 (World Map)

インタラクティブな世界地図を表示するWebアプリケーションです。マウスカーソルを国に合わせると、その国の名前が表示されます。 https://ocoe-puipui.github.io/map/

## 機能

- **インタラクティブな地図**: SVGベースの世界地図を表示
- **ホバー効果**: 国にマウスカーソルを合わせると、国名がリアルタイムで表示
- **検索機能**: 国名を検索してハイライト表示（英語・日本語対応）
- **クリック選択**: 国をクリックして選択・固定表示
- **美しいデザイン**: 
  - 海: 明るいブルー (`#bae6fd`)
  - 陸地: 自然なグリーン (`#22c55e`)
  - ホバー時: 濃いグリーン (`#15803d`) で強調
  - 検索結果: オレンジ色 (`#f97316`) でハイライト
- **レスポンシブデザイン**: 画面サイズに合わせて地図が自動調整

## 使い方

### 基本的な使い方
1. `index.html` をブラウザで開きます
2. 地図上の国にマウスカーソルを合わせます
3. 国名がツールチップに表示されます

### 検索機能
1. 検索ボックスに国名を入力します（英語または日本語）
2. 「検索」ボタンをクリックするか、Enterキーを押します
3. 該当する国がオレンジ色でハイライトされます
4. 「クリア」ボタンで検索結果を消去できます

### クリック選択
1. 国をクリックすると選択状態になります（オレンジ色）
2. 選択された国名が上部に固定表示されます
3. 海をクリックするか、同じ国を再度クリックすると選択解除されます

## 技術スタック

- **HTML5**: セマンティックなマークアップ
- **CSS3**: スタイリングとアニメーション
- **SVG**: ベクターベースの地図描画
