# 世界地図 (World Map)

インタラクティブな世界地図を表示するWebアプリケーションです。マウスカーソルを国に合わせると、その国の名前が表示されます。

## 機能

- **インタラクティブな地図**: SVGベースの世界地図を表示
- **ホバー効果**: 国にマウスカーソルを合わせると、国名がリアルタイムで表示
- **美しいデザイン**: 
  - 海: 明るいブルー (`#bae6fd`)
  - 陸地: 自然なグリーン (`#22c55e`)
  - ホバー時: 濃いグリーン (`#15803d`) で強調
- **レスポンシブデザイン**: 画面サイズに合わせて地図が自動調整

## 使い方

1. `index.html` をブラウザで開きます
2. 地図上の国にマウスカーソルを合わせます
3. 国名がツールチップに表示されます

## 技術スタック

- **HTML5**: セマンティックなマークアップ
- **CSS3**: スタイリングとアニメーション
- **SVG**: ベクターベースの地図描画

## カスタマイズ

### 色の変更

`index.html` 内のCSSセクションで色を変更できます：

```css
/* 海の色 */
svg {
    background-color: #bae6fd;
}

/* 陸地の色 */
path {
    fill: #22c55e;
}

/* ホバー時の色 */
path:hover {
    fill: #15803d;
}
```
